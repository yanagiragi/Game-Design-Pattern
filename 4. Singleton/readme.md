    2. 設定為類別的參考，讓物件可以容易被取得

        (繼承)     (用提供的介面注入)


Appendix

滿足以下擇一條件的話，可用 指定給類別的靜態成員：
    
    1. 產生物件的位置不確定
    2. 有多個地方可以生成物件
    3. 生成的位置無法引用到
    4. 有多種子類別
